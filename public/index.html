About this panel

This web interface uses the existing WhatsApp backend from the repository. Commands are executed through the bot engine in start/case.js, so WhatsApp remains the execution channel.

Use the Commands page to select a command, open its dedicated page, and fill the command fields with the right arguments.

Each command page now shows specific input fields, example usage, and notes for that command.

For raw command execution, use Home or the detail page with custom text.
